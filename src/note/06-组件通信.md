### 属性，父 -> 子

父传子：直接通过属性
子传父：子组件通过调用父组件传递的方法，将数据传给父组件

_父组件_

```vue
<script lang="ts" setup>
import { ref } from 'vue'
import Child from './Child.vue'

const car = ref('宝马')
const toy = ref('')

function getToy(val: string) {
  toy.value = val
}
</script>

<template>
  <div class="con">
    <h2>父组件</h2>
    <p>有辆车：{{ car }}</p>
    <p>来自子组件的玩具：{{ toy }}</p>
    <Child :car="car" :sendToy="getToy" />
  </div>
</template>
```

_子组件_

```vue
<script lang="ts" setup>
import { ref } from 'vue'

const toy = ref('奥特曼')

defineProps(['car', 'sendToy'])
</script>

<template>
  <div class="con">
    <h2>子组件</h2>
    <p>有个玩具：{{ toy }}</p>
    <p>来自父组件的车：{{ car }}</p>
    <button @click="sendToy(toy)">给父组件传数据</button>
  </div>
</template>
```

### 自定义事件，子 -> 父

_html的事件名用烤串法命名，驼峰法用在JS中_

传属性用 `:`, 传事件用 `@`

_父组件_

```vue
<script lang="ts" setup>
import { ref } from 'vue'
import Child from './Child.vue'

const toy = ref('')

function getToy(val: string) {
  toy.value = val
}
</script>

<template>
  <div class="con">
    <h2>父组件</h2>
    <p v-show="toy">来自子组件的玩具：{{ toy }}</p>
    <Child @send-toy="getToy" />
  </div>
</template>
```

_子组件_

```vue
<script lang="ts" setup>
import { ref } from 'vue'

const toy = ref('奥特曼')

const emit = defineEmits(['send-toy'])
</script>

<template>
  <div class="con">
    <h2>子组件</h2>
    <p>有个玩具：{{ toy }}</p>
    <button @click="() => emit('send-toy', toy)">给父组件传数据</button>
  </div>
</template>
```

### mitt，任意组件通信

1、下载mitt依赖

```bash
npm i mitt
```

2、添加到工具类中 utils/mitter

```js
import mitt from 'mitt'

export default mitt()
```

3、在要发送数据的组件中emit

```vue
<script lang="ts" setup>
import { ref } from 'vue'
import mitter from '@/utils/mitter'

const toy = ref('奥特曼')
</script>

<template>
  <div class="con">
    <h2>子组件</h2>
    <p>有个玩具：{{ toy }}</p>
    <button @click="() => mitter.emit('give-toy', toy)">把玩具给兄弟玩玩</button>
  </div>
</template>
```

4、在要接收数据的组件中on，不用时记得及时off

```vue
<script lang="ts" setup>
import { onUnmounted, ref } from 'vue'
import mitter from '@/utils/mitter'

const toy = ref('')

mitter.on('give-toy', (val: any) => {
  toy.value = val
})

onUnmounted(() => {
  mitter.off('give-toy')
})
</script>

<template>
  <div class="con">
    <h2>兄弟组件</h2>
    <p>玩具：{{ toy }}</p>
  </div>
</template>
```

### v-model底层原理，父 <=> 子

```vue
<template>
  <input v-model="inputValue" />
  <!-- 等价于 -->
  <input :value="inputValue" @input="inputValue = (<HTMLInputElement>$event.target).value" />
</template>
```

若是自己要封装一个输入组件，且支持v-model呢

```vue
<!-- 正常使用 -->
<MyInput v-model="inputValue" />
<!-- 底层 -->
<MyInput :modelValue="inputValue" @update:modelValue="inputValue = $event" />
```

_自定义的 MyInput 组件_

```vue
<script lang="ts" setup>
defineProps(['modelValue'])
const emit = defineEmits(['update:modelValue'])
</script>

<template>
  <input
    :value="modelValue"
    @input="emit('update:modelValue', (<HTMLInputElement>$event.target).value)"
  />
</template>
```

### $attrs, 祖 -> 孙

父组件传给子组件的属性，子组件声明接收的在props中，未接收的在`$attrs`中

v-bind的使用 `:a="name"` 和 `v-bind="{a:name}"`相同

父组件

```vue
<script lang="ts" setup>
import { ref } from 'vue'
import Child from './Child.vue'

const msg = ref('hello world 1630')

function editMsg(val: string) {
  msg.value = val
}
</script>

<template>
  <div class="con">
    <h2>父组件</h2>
    <p>有一条信息：{{ msg }}</p>
    <Child v-bind="{ msg, editMsg }" />
  </div>
</template>
```

子组件

```vue
<script lang="ts" setup>
import GrandChild from './GrandChild.vue'
</script>

<template>
  <div class="con">
    <GrandChild v-bind="$attrs" />
  </div>
</template>
```

孙组件

```vue
<script lang="ts" setup>
defineProps(['msg', 'editMsg'])
</script>

<template>
  <div class="con">
    <h2>孙组件</h2>
    <p>收到来自祖组件的消息：{{ msg }}</p>
    <button @click="editMsg('hi grandfather ~~~')">孙组件修改祖组件</button>
  </div>
</template>
```

### $refs, 父 -> 子；$parent, 子 -> 父

父组件

```vue
<script lang="ts" setup>
import { ref } from 'vue'
import Child from './Child.vue'
import Child2 from './Child2.vue'

const house = ref(4)
const c1 = ref()
const c2 = ref()

function changeToy() {
  c1.value.toy = '小猪佩奇'
}

function changeComputer() {
  c2.value.computer = '华为'
}

function addBook(refs: any) {
  for (let key in refs) {
    refs[key].book += 2
  }
}

defineExpose({ house })
</script>
```

子组件1

```vue
<script lang="ts" setup>
import { ref } from 'vue'

const toy = ref('奥特曼')
const book = ref(3)

function changeHouse(parent: any) {
  parent.house -= 1
}

defineExpose({ toy, book })
</script>

<template>
  <div class="con">
    <h2>子组件</h2>
    <p>玩具：{{ toy }}</p>
    <p>数据：{{ book }} 本</p>
    <button @click="changeHouse($parent)">修改父亲的房产</button>
  </div>
</template>
```

子组件2

```vue
<script lang="ts" setup>
import { ref } from 'vue'

const computer = ref('联想')
const book = ref(6)

defineExpose({ computer, book })
</script>

<template>
  <div class="con">
    <h2>兄弟组件</h2>
    <p>电脑：{{ computer }}</p>
    <p>书籍：{{ book }} 本</p>
  </div>
</template>
```

### provide & inject, 祖 -> (多重)孙，不用经过中间人

祖组件

```vue
<script lang="ts" setup>
import { provide, reactive, ref } from 'vue'
import Child from './Child.vue'

const money = ref(100)
const car = reactive({ brand: '宝马', price: 30 })

function updateMoney(val: number) {
  money.value -= val
}

provide('moneyContext', { money, updateMoney })
provide('car', car)
</script>

<template>
  <div class="con">
    <h2>父组件</h2>
    <p>存款：{{ money }}</p>
    <p>汽车：一辆{{ car.brand }}，价值{{ car.price }} W</p>
    <Child />
  </div>
</template>
```

孙组件

```vue
<script lang="ts" setup>
import { inject } from 'vue'

const { money, updateMoney } = inject('moneyContext', { money: 0, updateMoney: (n: number) => {} })
const car = inject('car', { brand: '默认品牌', price: 0 })
</script>

<template>
  <div class="con">
    <h2>孙组件</h2>
    <p>收到来自祖组件的存款：{{ money }}</p>
    <p>汽车：一辆{{ car.brand }}，价值{{ car.price }} W</p>
    <button @click="updateMoney(6)">孙组件修改祖组件</button>
  </div>
</template>
```
